# AI-Powered Autonomous Indoor Inspection & Assistance Robot

## Project Overview
This project aims to develop an AI-powered autonomous indoor robot capable of performing inspection and assistance tasks using ROS 2, SLAM, Navigation2, and Computer Vision.

## Features
- Autonomous Navigation
- Indoor Mapping using SLAM
- Localization
- Path Planning
- Obstacle Avoidance
- Object Detection (YOLO)
- Computer Vision (OpenCV)
- Inspection Report Generation

## Technology Stack
- ROS 2
- Ubuntu
- Python
- C++
- OpenCV
- YOLOv8
- SLAM Toolbox
- Navigation2
- Gazebo
- RViz2
